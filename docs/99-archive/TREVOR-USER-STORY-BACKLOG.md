## User Story Backlog (Prioritized)

## Epic A - Core Platform (MVP)

### 1. Dashboard View
As a user, I can view a dashboard with active onboardings and at-risk clients
**AC:** Dashboard displays counts, clicking at-risk opens filtered list

### 2. Pipeline Kanban
As a user, I can view and move clients on a Kanban pipeline
**AC:** 6 columns, drag/drop persists and creates StageEvent

### 3. Add Client
As a user, I can add a new client with required fields
**AC:** Form validation, client appears in Pipeline & List

### 4. Authentication
As a user, I can authenticate and be scoped to my agency
**AC:** JWT issued, cross-agency access rejected

## Epic B - Integrations & Sync (v1)

### 5. Slack Integration
As admin, I can connect Slack and authorize message sync

### 6. Gmail Integration
As user, I can connect Gmail and view recent threads

### 7. Ads Integration
As user, I can connect Google/Meta Ads for hourly KPIs

## Epic C - AI Assistant & RAG (v1)

### 8. Health Summary
As user, I can ask assistant to summarize client health

### 9. Email Draft
As user, I can ask assistant to draft email (no auto-send)

### 10. Document Upload
As user, I can upload documents for RAG indexing

## Epic D - Tickets & Knowledge (v2)

### 11. Support Tickets
Support ticket Kanban with sentiment

### 12. Knowledge Base
Knowledge Base search with SOP cards

## Epic E - Automations (v2)

### 13. Workflows
Create IF/THEN workflows with triggers and actions
